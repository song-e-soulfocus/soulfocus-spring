# 🧠 SoulFocus

포스텔러처럼 사용자의 심리와 성향을 가볍게 진단해주는 **웹 기반 심리 테스트 서비스**입니다.  
여러 가지 질문에 답하면, 사용자의 성향을 분석하여 결과를 카드 형태로 보여줍니다.

---

## 📌 프로젝트 개요

- **프로젝트 이름**: SoulFocus
- **목표**  
  - 사용자가 재미있게 할 수 있는 심리/성향 테스트 제공  
  - 결과를 통해 자신의 성향을 한눈에 이해할 수 있도록 시각화  
- **타겟**  
  - 가볍게 심리테스트를 즐기고 싶은 사용자  
  - 친구/연인과 결과를 공유하며 놀고 싶은 사람들
